## 🌟 Features

- ⚡️ **Real-time Performance Tracking**
  - Instant WPM (Words Per Minute) calculation
  - Live accuracy monitoring
  - Dynamic progress visualization

- 🎮 **Game Modes**
  - Multiple difficulty levels
  - Race against AI opponents
  - Practice mode
  - Daily challenges

- 📊 **Statistics & Analytics**

  - Progress charts

  - Cross-browser compatibility

## 🚀 Quick Start

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/typing-race.git
cd typing-race
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start development server
```bash
npm run dev
# or
yarn dev
```

4. Build for production
```bash
npm run build
# or
yarn build
```

## 🎯 Usage

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Enter` | Start new race |
| `Escape` | Reset current race |
| `Tab` | Switch theme |
| `Space` | Pause/Resume |
